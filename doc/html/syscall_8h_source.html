45 /* The system call interface. These are the operations the Nachos
46  * kernel needs to support, to be able to run user programs.
47  *
48  * Each of these is invoked by a user program by simply calling the
49  * procedure; an assembly language stub stuffs the system call code
50  * into a register, and traps to the kernel. The kernel procedures
51  * are then invoked in the Nachos kernel, after appropriate error checking,
52  * from the system call entry point in exception.cc.
53  */
